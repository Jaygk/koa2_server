# koa2_server
> 用koa2实现的简单的restful风格的crud 服务端api

## Project setup

```
npm install
```

## Run server

```
node ./src/app.js
```

## Api

- ###### '/api/students':

  + type: get
  + 获取学生信息列表

- ###### '/api/add':

  + type: post
  + 添加一条学生信息

- ###### '/api/edit': 

  + type: put
  + 修改一条学生信息

- ###### '/api/delete':

  + type: delete
  + 删除一条学生信息
